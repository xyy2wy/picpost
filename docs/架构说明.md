# 项目架构说明

## 目录结构

```
picpost/
├── main.py                          # CLI 入口
├── web_app.py                       # Web 入口 (streamlit run web_app.py)
├── config.yaml                      # 用户配置
├── requirements.txt                 # Python 依赖
│
├── core/                            # 核心业务逻辑
│   ├── __init__.py
│   ├── constants.py                 # 常量定义（元素名称/值、位置、颜色、状态枚举）
│   ├── config.py                    # Config 类：读写 config.yaml
│   ├── container.py                 # ImageContainer：封装图片 + EXIF 元数据
│   └── processors.py               # 处理器链（Composite 模式）：所有布局/效果处理器
│
├── services/                        # 功能服务（纯函数为主，输入 Image → 输出 Image）
│   ├── __init__.py
│   ├── color.py                     # 调色 / 滤镜 / 自动增强 / 防盗水印
│   ├── annotation.py                # 文字贴纸 / 图上标注
│   ├── compose.py                   # 长图拼接 / 前后对比
│   ├── cover.py                     # 首图文字卡片
│   ├── processing.py                # 处理管线：构建 ProcessorChain、批量处理
│   ├── publish.py                   # 发布草稿打包
│   ├── selection.py                 # 选片服务（缩略图、过滤、导出）
│   ├── smart_crop.py                # 智能裁切（人脸优先 / 显著性）
│   ├── video.py                     # FFmpeg 视频生成
│   └── xiaohongshu.py               # 小红书工具：切图 / 拼图 / 页码 / 多比例
│
├── ai/                              # AI 能力层（可选依赖 openai）
│   ├── __init__.py
│   ├── preset.py                    # AI 预设建议 / 选图 / 风格标签 / 文案生成
│   └── style_pack.py                # 风格包 schema / 序列化 / 校验
│
├── cli/                             # CLI 交互层
│   ├── __init__.py
│   ├── menu.py                      # 菜单组件系统（Composite 模式）
│   ├── setup.py                     # 配置加载、菜单组装（数据驱动）
│   └── xiaohongshu.py               # 小红书多图工具的 CLI 交互处理函数
│
├── web/                             # Web 前端层（Streamlit）
│   ├── __init__.py
│   ├── preview.py                   # 实时预览生成
│   └── app.py                       # Streamlit 主应用
│
├── utils_pkg/                       # 工具函数
│   ├── __init__.py                  # 统一导出接口
│   └── helpers.py                   # ExifTool、图片缩放/拼接/裁切、文件操作
│
├── fonts/                           # 字体资源
├── logos/                           # Logo 资源
├── images/                          # 示例图片
├── input/                           # 输入图片（用户放入）
├── output/                          # 输出图片（处理后）
├── app_data/                        # 应用数据（预设、历史）
├── scripts/                         # 构建 / 安装脚本
│   ├── install.sh / install.ps1     # 初始化安装
│   ├── start.applescript            # macOS 启动脚本
│   ├── main.spec                    # PyInstaller 打包配置
│   └── build_win_pkg.spec           # Windows 发布包配置
├── docs/                            # 文档
└── .github/                         # CI/CD
```

## 分层关系

```
main.py / web_app.py (入口)
    ↓
cli/ 或 web/  (交互层 —— 负责 I/O 交互)
    ↓
services/     (服务层 —— 业务逻辑编排)
    ↓
core/         (核心层 —— 数据模型、处理器、配置)
    ↓
utils_pkg/    (工具层 —— 无业务语义的通用工具)
```

## 兼容性

旧的 `entity/`、`enums/` 目录及根目录的 `*_service.py` 文件已删除。
所有代码统一使用新路径（`core.*`、`services.*`、`ai.*` 等）。

## 开发指南

- **新增功能服务**：放在 `services/` 下，保持纯函数风格
- **新增 AI 能力**：放在 `ai/` 下
- **新增 CLI 命令**：在 `cli/xiaohongshu.py` 添加函数，并在 `cli/setup.py` 注册菜单项
- **新增处理器**：在 `core/processors.py` 添加类，并在 `services/processing.py` 注册到处理链
